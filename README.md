- `GET /`
- create a [middleware](https://expressjs.com/en/guide/using-middleware.html#using-middleware) logger
- index `GET /sharks`
- show `GET /sharks/:id`
- create `POST /sharks`
- update `PATCH /sharks/:id`
- destroy `DELETE /sharks/:id`
